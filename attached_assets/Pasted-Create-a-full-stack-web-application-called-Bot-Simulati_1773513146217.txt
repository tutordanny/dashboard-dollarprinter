BOT SIMULATION ENGINE

Create a file called bot-engine.js.

This engine must simulate trading without connecting to any external broker API.

The engine must:

Generate fake market price ticks
Execute bot trades
Determine win/loss outcome
Update balance
Store trades in database

Example engine logic:

generate random price movement
evaluate bot entry condition
calculate result

Trade outcome logic example:

if randomNumber < winProbability
trade = win
profit = stake * payoutRatio
else
trade = loss
profit = -stake
